# Binary-Search-Tree
## 1-) [7, 5, 1, 8, 3, 6, 0, 9, 4, 2] dizisinin Binary-Search-Tree aşamalarını yazınız.
* ROOT Değeri = 5
* 7 root değerinden büyük olduğu için root'un sağında yer alır.
* 2 root değerinden küçük olduğu için root'un solunda yer alır.
```
           5
        /     \
       2        7
      / \      / \
     1    3   6   8
    /      \       \
   0        4       9
```
